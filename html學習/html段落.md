# HTML段落
```
HTML<p>元素定義一個段落。

段落始終以換行開頭，瀏覽器會在段落前後自動添加一些空格（邊距）。

例子
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
```
# HTML顯示
```
您無法確定HTML的顯示方式。

大屏幕或小屏幕以及調整大小的窗口都會產生不同的結果。

使用HTML，您無法通過在HTML代碼中添加額外的空格或額外的行來更改顯示。

顯示頁面時，瀏覽器將自動刪除所有多餘的空格和行：

例子
<p>
This paragraph
contains a lot of lines
in the source code,
but the browser
ignores it.
</p>

<p>
This paragraph
contains         a lot of spaces
in the source         code,
but the        browser
ignores it.
</p>
```
# HTML水平規則
```
該<hr>標籤定義HTML頁的專題突破，並且大多數情況下顯示為一條水平線。

該<hr>元件用於分離內容（或定義的變化）在HTML頁中：

例子
<h1>This is heading 1</h1>
<p>This is some text.</p>
<hr>
<h2>This is heading 2</h2>
<p>This is some other text.</p>
<hr>
```
# HTML換行符
```
HTML<br>元素定義一個換行符。

使用<br>，如果你想有一個換行符（新行），而無需啟動一個新的段落：

例子
<p>This is<br>a paragraph<br>with line breaks.</p>

```
# HTML <-pre->元素
```
HTML<pre>元素定義了預格式化的文本。

<pre>元素內的文本以固定寬度的字體（通常為Courier）顯示，並且保留空格和換行符：

例子
<pre>
  My Bonnie lies over the ocean.

  My Bonnie lies over the sea.

  My Bonnie lies over the ocean.

  Oh, bring back my Bonnie to me.
</pre>
```

# HTML格式元素
```
格式化元素旨在顯示特殊類型的文本：
<b> - 粗體文字
<strong> -重要文字
<i> -斜體文字
<em> -強調文字
<mark> -標記文字
<small> -較小的文字
<del> -刪除的文字
<ins> -插入文字
<sub> -下標文字
<sup> -上標文字
```
# HTML <-b->和<-strong->元素
```
HTML<b>元素定義了粗體文本，沒有任何額外的重要性。

例子
<b>This text is bold</b>

HTML<strong>元素定義了非常重要的文本。裡面的內容通常以粗體顯示。

例子
<strong>This text is important!</strong>
```
# HTML <-i->和<-em->元素
```
HTML<i>元素以替代的語音或語氣定義了文本的一部分。裡面的內容通常以斜體顯示。

提示：該<i>標籤通常用來表示一個技術術語，從另一種語言，一個想法，一個船名等短語

例子
<i>This text is italic</i>
HTML<em>元素定義強調的文本。裡面的內容通常以斜體顯示。

提示：屏幕閱讀器會<em> 使用語言重讀來重點發音這些單詞。

例子
<em>This text is emphasized</em>
```
# HTML <-  small  ->元素

```
例子
<small>This is some smaller text.</small>
```
# HTML <-mark->元素
```
HTML<mark>元素定義應標記或突出顯示的文本：

例子
<p>Do not forget to buy <mark>milk</mark> today.</p>
 ```
# HTML <-del->元素
```
HTML<del>元素定義了已從文檔中刪除的文本。瀏覽器通常會在刪除的文本後加上一行：

例子
<p>My favorite color is <del>blue</del> red.</p>
```
# HTML <-ins->元素
```
HTML<ins>元素定義了已插入文檔中的文本。瀏覽器通常會在插入的文本下劃線：

例子
<p>My favorite color is <del>blue</del> <ins>red</ins>.</p>
```
# HTML <-sub->元素
```
HTML<sub>元素定義下標文本。下標文字顯示在正常行下方半個字符，有時以較小的字體顯示。下標文本可用於化學式，例如H 2 O：

例子
<p>This is <sub>subscripted</sub> text.</p>
```
# HTML <-sup->元素
```
HTML<sup>元素定義上標文本。上標文字顯示在法線上方半個字符，有時以較小的字體呈現。上標文本可用於腳註，例如WWW [1]：

例子
<p>This is <sup>superscripted</sup> text.</p>
```
# HTML文本格式元素
```
Tag	             Description
<b>	             Defines bold text
<em>	           Defines emphasized text 
<i>	             Defines a part of text in an alternate voice or mood
<small>	         Defines smaller text
<strong>	       Defines important text
<sub>	           Defines subscripted text
<sup>	           Defines superscripted text
<ins>	           Defines inserted text
<del>	           Defines deleted text
<mark>	         Defines marked/highlighted text
```
